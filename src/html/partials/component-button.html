Button

버튼은 사용자가 클릭하거나 터치 할 때 발생할 동작을 나타냅니다.

  • 버튼은 3개 크기로 구분합니다.
    이 이외의 버튼 스타일은 사용하지 않습니다. 각 버튼의 설명을 숙지하고 목적에 맞게 사용하도록합니다.
  • 버튼에 아이콘을 조합하여 사용할 경우 버튼 텍스트를 span태그로 감싸 button__text 라는 클래스를 추가해 사용합니다.
Primary Button

화면에서 가장 주요한 액션에 사용하는 버튼입니다. 한 화면에 한번만 사용하는걸 기본으로 합니다.

Size: lg

Size: md

Size: sm

primary와 default 버튼의 중간 정도 위계를 갖는 액션에 사용합니다.

Size: md

Size: sm
